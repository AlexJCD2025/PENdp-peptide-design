# Ponceau S / NC膜 短肽检测假阴性排查指南

> 常见客户投诉场景：定制多肽在NC膜上经Ponceau S染色看不到斑点，客户质疑多肽质量。
> 根因通常是检测方法学不匹配，而非多肽本身不合格。

---

## 一、根因速览

| 因素 | 说明 |
|:-----|:------|
| 短肽(<3kDa) NC膜保留差 | 洗涤过程中容易穿膜或流失 |
| Ponceau S检测下限 | 全长蛋白~200ng；短肽因膜保留差+结合位点少，实际需求达μg级 |
| 序列依赖性强 | Ponceau S靠正电荷残基(Lys/Arg/His)+疏水区结合；酸性/亲水短肽染色极弱 |
| 膜孔径问题 | 0.45μm NC膜完全不适合<10kDa肽；应换0.22μm PVDF |

## 二、行业质控标准说明

研究级定制多肽的行业标准放行标准 = **MS分子量 + HPLC纯度**

| 验证项 | 能证明 | 不能证明 |
|:-------|:-------|:---------|
| MS分子量 | 主峰分子量符合理论值 | 下游实验功能、溶解度、膜结合能力、生物活性、构象 |
| HPLC纯度 | 主峰面积占比 | 同上，以及绝对含量(需AAA/qNMR) |

**话术核心**："MS+HPLC合格说明化学身份和纯度符合交付标准，但Ponceau S/NC膜点样不是适合短肽的检测方法，不能作为多肽不存在的证据。"

## 三、给客户的替代方案（优先级排序）

| 优先级 | 方法 | 说明 |
|:------:|:-----|:------|
| 🥇 | **LC-MS/HPLC检测复溶液** | 最直接，10分钟出结果 |
| 🥈 | Tricine-SDS-PAGE + 银染 | 比普通glycine体系更适合短肽 |
| 🥈 | 0.22μm PVDF膜点样 | 甲醇预润湿，风干固定后染色 |
| 🥉 | UV214检测 | 肽键吸收，不受序列影响 |
| ❌不推荐 | BCA/Bradford/UV280 | 对短肽灵敏度极差或无响应 |

## 四、回答客户的三段式话术

```
第一层（技术解释）：
"这条多肽约1-3kDa，属于短肽范围。Ponceau S染色+NC膜点样对短肽的保留和染色灵敏度都有限——并非多肽不存在或质量有问题，而是这个检测体系不太适合短肽。"

第二层（替代方案）：
"建议取同一管复溶后的点样液直接跑LC-MS验证。也可以试试0.22μm PVDF膜/Tricine-SDS-PAGE+银染。"

第三层（后续跟进）：
"我们把原始QC图谱和推荐溶解方案发您参考。如果复测后仍有问题，可以寄回少量样品，我们免费补做LC-MS复检。"
```

## 五、内部流程改进

对<5kDa短肽订单，交付时在COA中注明：
> "短肽不建议用Ponceau S/普通NC膜作为存在性检测；推荐LC-MS/HPLC/UV214/AAA等方法。"
