## 核心概念更新

### 关键认知转折
```
Nat Commun 2020: "五环三萜替代CHOL → 效果差" （Group III结论）
        ↓
Nat Commun 2025: "熊果酸添加20% → 40倍提升" （苗蕾）
        ↓
结论: "替代"与"添加"是两种完全不同的策略！
        ↓
对欣肽AA-LNP启示:
  - 替代路线(10% AA换CHOL) ✅ 可行，已验证83.9%包封率
  - 添加路线(AA作为第五组分) → 新方向，参考苗蕾20%比例
```

### AA vs UA vs CA 同分异构体关系
- 积雪草酸(AA)、熊果酸(UA)、齐墩果酸(OLA)、科罗索酸(CA) → 分子式C30H48O5，互为同分异构体
- 但C-3位构型不同→LNP中表现差异显著
- CA可完全替代，UA添加效果最优，AA需要进一步验证添加策略
