# 四条多肽的SPPS固相合成方案

**日期：** 2026-05-27
**适用范围：** T7（脑靶向）、iRGD（环化·肺靶向）、iRGD-L（线性·肺靶向）、6R（电荷·肺靶向）
**通用链路：** LNP ← DSPE ← PEG ← Maleimide ← Cys(多肽)

---

## 1. T7 — HAIYPRH-G-C（脑靶向，TfR）

| 属性 | 值 |
|:----|:---|
| 序列 | H-A-I-Y-P-R-H-G-C |
| 长度 | 9aa（线性） |
| 靶点 | TfR（转铁蛋白受体，BBB跨细胞） |
| D9评分 | 7.0（C端Cys +2.0）|
| D1评分 | 9.0（精确匹配T7基序）|
| D6评分 | 8.5（合成容易）|

### 合成方案

| 步骤 | 条件 |
|:----|:-----|
| 树脂 | Rink Amide（C端酰胺化，增强血清稳定性） |
| 偶联剂 | HBTU/HOBt/DIEA（标准），P-R位置用HATU |
| 偶联时间 | 标准30min，P-R延长至60min |
| 脱保护 | TFA:TIS:H₂O:EDT = 92:2.5:2.5:3, 2.5h |
| 纯化 | RP-HPLC, C18, 0-60% ACN/0.1% TFA |
| 预期产率 | 粗肽纯度>90%，总产率>85% |

### 偶联条件
- 粗肽溶于pH 6.5 PBS（0.1M磷酸盐）
- 与DSPE-PEG2000-Maleimide按1:1.2摩尔比
- 室温2h，N₂保护
- 超滤（10kDa cutoff）去除未反应肽

## 2. iRGD（环化）— C-RGDKGPDC-G-C（肺靶向，αvβ3）

| 属性 | 值 |
|:----|:---|
| 序列 | C-R-G-D-K-G-P-D-C-G-C（11aa）|
| 结构 | C1-C2二硫键环化，C3游离 |
| 靶点 | αvβ3/β5整合素 |
| 关键工艺 | 正交保护选择性环化（C1-C2成环，C3保留）|
| D1评分 | 9.5（环化RGD αvβ3高亲和）|
| D9评分 | 7.0（C端Cys +2.0）|
| D6评分 | 7.5（11aa）|
| 预期产率 | ~60% |

### 正交保护环化工序

```
保护策略：C1(Trt) ... C2(Trt) ... C3(Acm)

Step 1: Fmoc固相合成线性前体
  从C端到N端逐一偶联：
  C3(Acm)-G-C2(Trt)-D(OtBu)-P-G-K(Boc)-G-D(OtBu)-R(Pbf)-G-C1(Trt)
  
Step 2: 全脱保护（保留Acm）
  95% TFA + 2.5% TIS + 2.5% H₂O, 2.5h
  脱除：Trt, Boc, Pbf, OtBu
  保留：Acm（C3保持保护状态）

Step 3: 环化C1-C2
  DMSO/H₂O (1:9)，pH 7.0（0.1M NH₄HCO₃）
  室温搅拌12-24h，多肽浓度~0.5mg/mL（低浓度减少二聚）
  TLC或HPLC监测反应

Step 4: RP-HPLC纯化环化产物
  收集环化产物峰（环化后保留时间通常前移）

Step 5: 脱Acm → 释放C3-SH
  I₂ (10 eq) / 90% AcOH, 室温2h
  抗坏血酸淬灭I₂
  G25脱盐柱除I₂

Step 6: 冻干 → 用于LNP偶联
```

### LNP偶联注意事项
- C3游离-SH极易氧化 → 纯化后立即使用或冻存于pH 5.0柠檬酸缓冲液
- 偶联前加1-2 eq TCEP（三(2-羧乙基)膦），室温15min还原
- 脱盐去除TCEP后再加DSPE-PEG-Maleimide
- 反应在pH 6.5 PBS中，N₂保护，室温2h

### 备选简化方案（免环化）
- 直接用原序列CRGDKGPDC中的Lys（ε-NH₂）偶联NHS-PEG-DSPE
- 不环化，不保护，直接反应
- D9评分降至5.0（方向性差），但节省~2天

## 3. iRGD-L（线性）— CRGDKGPD-G-C（肺靶向对照）

| 属性 | 值 |
|:----|:---|
| 序列 | C-R-G-D-K-G-P-D-G-C（10aa，不环化）|
| 结构 | 线性，无二硫键 |
| 用途 | iRGD环化版的阴性对照 |
| D1评分 | 8.5（RGD基序命中的基础分，无环化加分）|
| D9评分 | 7.0（C端Cys +2.0）|
| 预期产率 | ~75% |

### 合成方案

```
C ─ R ─ G ─ D ─ K ─ G ─ P ─ D ─ G ─ C
↑N端    RGD基序(线性，柔性)          ↑偶联

合成：标准Fmoc-SPPS
  - 树脂：Rink Amide
  - 偶联剂：HBTU/HOBt/DIEA
  - 脱保护：TFA:TIS:H₂O = 95:2.5:2.5, 2.5h
  - 纯化：RP-HPLC (C18)
  - 无环化步骤 → 1-2天完成

与环化版的关键差异：
  ✗ 不需要Acm保护
  ✗ 不需要DMSO氧化环化
  ✗ 不需要I₂脱保护
  ✓ 偶联直接：C端Cys直接对Maleimide
```

### 与iRGD环化版的对比定位

| 对比项 | iRGD（环化） | iRGD-L（线性） |
|:-------|:------------|:---------------|
| αvβ3亲和力 | IC50 ~nM | IC50 ~μM（低10-50倍）|
| RGD构象 | 固定loop | 自由旋转 |
| 合成耗时 | 3-4天 | 1-2天 |
| 预期产率 | ~60% | ~75% |

## 4. 6R — RRRRRR-G-C（肺靶向，电荷介导）

| 属性 | 值 |
|:----|:---|
| 序列 | R-R-R-R-R-R-G-C（8aa）|
| 靶向机制 | 阳离子电荷→蛋白冠吸附→肺泡毛细血管富集 |
| 合成难度 | ⚠️ **高** — Poly-Arg严重的链聚集 |
| D1评分 | 5.0（无已知靶向基序）|
| D9评分 | 7.0（C端Cys +2.0）|
| 预期产率 | ~40%（自己合成）|

### 合成难点

6个连续Arg形成分子间β-sheet，导致：
1. 偶联效率逐残基下降（第5-6个Arg可能<50%）
2. 粗肽纯度~70%（正常序列>90%）
3. 总产率~40%

### 缓解策略

| 措施 | 具体操作 |
|:----|:---------|
| 偶联剂 | 用HATU/DIEA（HBTU不够强）|
| 溶剂 | NMP/DMSO = 1:1 混合溶剂，替代纯DMF |
| 温度 | 50°C微波辅助 |
| 偶联时间 | 45min/次（标准15min×3）|
| 监测 | 每步Kaiser test |
| 脱保护 | TFA:TIS:H₂O:EDT = 92:2:2:4, 3h（必须加EDT清除Pbf）|

### 纯化注意事项
- 6个Arg带6个正电荷 → HPLC峰易拖尾
- 建议用宽孔柱（300Å）或离子对试剂（0.1% HFBA）
- 先用G25脱盐柱，再用RP-HPLC

### 建议
Poly-Arg最划算的方案是**外包给多肽合成公司**（CPC Scientific/金斯瑞/中科新生命等）：
- 6-11aa短肽：300-800元/条
- 纯度 >95%（比自己做的~70%高很多）
- 省时省力，批次一致性好
