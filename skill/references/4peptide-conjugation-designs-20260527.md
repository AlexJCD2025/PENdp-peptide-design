# 四种多肽的LNP偶联设计与合成方案

**日期**: 2026-05-27 | **来源**: PENdp分子机制讨论

## 偶联化学基础

所有多肽通过 Cys-Maleimide 反应连接到 DSPE-PEG 锚定的 LNP 表面：

```
LNP脂双层 → DSPE(锚定) → PEG(间隔臂) → Maleimide(反应基团)
                                                ↓
                                          Cys-SH(多肽)
```

## 四种序列的设计对比

### 1. T7（脑靶向 — TfR受体）
- **序列**: HAIYPRH（7aa，线性）
- **推荐设计**: HAIYPRH-G-C（C端加Gly-Cys）
- **D9评分**: 7.0（C端Cys +2.0）
- **合成难度**: 低（产率>85%）
- **偶联方式**: C端Cys直接对Maleimide

### 2. iRGD（环化 — 肺靶向 αvβ3）
- **序列**: CRGDKGPDC（9aa，两端Cys二硫键环化）
- **约束**: 两个Cys已用于环化 → 无游离Cys做偶联
- **推荐设计**: C-RGDKGPDC-G-C（11aa，加第三Cys）
- **正交保护**: C1/C2用Trt保护（DMSO环化），C3用Acm保护（I₂脱保护）
- **D9评分**: 7.0（从6.0提升）
- **合成难度**: 中（产率~60%，需两步环化+纯化）

### 3. iRGD-L（线性 — 肺靶向 αvβ3，对照用）
- **建议序列**: CRGDKGPD-G-C（去掉末端Cys）
- **与环化版差异**: RGD loop自由旋转 → 亲和力低10-50倍（文献数据）
- **D1评分**: 8.5（RGD命中，无环化加分）
- **合成难度**: 低-中（产率~75%，无环化步骤）
- **定位**: 环化版的阴性对照，验证"环化对靶向效率的贡献"

### 4. 6R（阳离子 — 肺靶向·电荷介导）
- **序列**: RRRRRR（6aa，线性）
- **约束**: 无Cys（Maleimide无法反应），无Lys（NHS无法反应）
- **推荐设计**: RRRRRR-G-C（8aa）
- **Poly-Arg聚集**: 6个连续Arg形成分子间β-sheet，需要升温+强活化剂(HATU)+混合溶剂(NMP/DMSO)
- **合成难度**: 高（Poly-Arg最难合成，产率~40%）
- **建议**: 外包比自合成更经济

## 四者对比

| 维度 | T7 | iRGD(环) | iRGD-L(线) | 6R |
|:----|:---|:---------|:-----------|:---|
| 靶向机制 | TfR受体 | αvβ3整合素 | αvβ3(弱) | 蛋白冠→αvβ3 |
| 总长 | 9aa | 11aa | 10aa | 8aa |
| 环化 | 无 | S-S(C1-C2) | 无 | 无 |
| D1 | 9.0 | 9.5 | 8.5 | 5.0 |
| D9 | 7.0 | 7.0 | 7.0 | 7.0 |
| 难度 | 低 | 中 | 低-中 | 高 |
| 推荐顺序 | ① | ② | ②'(对照) | ③(外包) |
