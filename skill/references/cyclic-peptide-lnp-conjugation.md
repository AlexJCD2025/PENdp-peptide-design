# 环化多肽-LNP偶联机制

**2026-05-27** | PENdp D9评分设计依据

## 核心化学：Cys-Maleimide 偶联

PENdp 的 LNP 表面修饰标准方案：

```
LNP脂双层 → DSPE(磷脂锚定) → PEG(间隔臂, 2k-5k) → Maleimide(反应基团)
                                                          |
                                                   Cys-SH(多肽巯基)
```

Maleimide 与游离 Cys 的巯基(-SH)形成 **稳定的硫醚键**（不可逆），pH 6.5-7.5 时反应速率最高。

## 环化多肽的特殊挑战

像 iRGD（**C**RGDKGPD**C**）这种两端 Cys 环化的分子，两个 Cys 已形成二硫键（-S-S-）用来维持环状构象，**不能再作为偶联位点**。

## 三种偶联策略

### 方案一：第三颗 Cys（最常用）

```
   C ---- RGDKGPDC ---- G ---- C（额外）
   ↑环化用                    ↑偶联用
```

在序列末端额外加一个游离 Cys，专门用于 maleimide 偶联。环化用 Cys 和偶联用 Cys 各自独立，互不干扰。

**优点：** 设计简单，方向确定（外端暴露）
**缺点：** 序列多一个残基，可能影响合成产率

### 方案二：一端偶联 + 另一端环化

```
   C ---- RGD ---- C
   ↑偶联             ↑环化
```

只用一个 Cys 做偶联（通常是 N 端），另一个 Cys 与内部残基或另一端形成二硫键。但这要求环化方式不是简单的"两端二硫键"。

### 方案三：非 Cys 环化替代

- **Lactam 桥（酰胺键）** — Lys 侧链-NH₂ 与 Asp/Glu 侧链-COOH 形成酰胺环
- **点击化学（azide-alkyne）** — 叠氮+炔基的 CuAAC 或 SPAAC 环化
- **苯环/碳桥** — 非天然氨基酸连接

释放 C 端或 N 端的 Cys 专门给 maleimide 偶联。

## D9 评分逻辑

D9（偶联定向性）的评分直接基于上述化学：

| 条件 | 得分 | 化学原理 |
|:----|:----:|:---------|
| **C端Cys** | **+2.0** | 最优——LNP表面向外展示，靶向基序最暴露 |
| **N端Cys** | **+1.5** | 良好——但基序可能被PEG覆盖，影响结合 |
| **两端Cys环化** | **-0.5** | Cys数量够但方向不确定 |
| **内部Lys ≥ 2** | **-1.0** | 多个NH₂位点→产物混合物 |
| **序列 ≤ 6aa** | **+1.0** | 短序列过程控制更好 |

**结论：** D9 评分最高的设计是 C 端额外游离 Cys + 内部两个 Cys 做环化。

## 真实案例

| 肽 | 序列 | 修饰方式 |
|:---|:-----|:---------|
| T7（脑靶向） | C**HAIYPRH** | N端Cys直接偶联DSPE-PEG2K |

| TAT-DSPE（CPP） | GRKKRRQRRRPPQ**C** | C端加Cys专门偶联 |

## 替代偶联化学

除了 Cys-maleimide，PENdp 也跟踪以下方法：

- **NHS-ester + Lys** — 与 Lys 侧链-NH₂ 反应，但选择性差（多 Lys 时产物混杂）
- **Click chemistry（azide-DBCO）** — 无铜点击，生物正交，不依赖 Cys
- **C18/Chol 锚定** — 多肽直接接脂质尾巴，无需 PEG 连接子
- **Sortase A 酶促连接** — 位点特异性，但酶成本高
