## 一、mRNA 操作部分（AA-LNP-mRNA）

### 1.1 技术路线确认

AA-LNP = 积雪草酸（Asiatic Acid）替代胆固醇的LNP体系

| 参数 | 确定值 | 备注 |
|------|--------|------|
| 可电离脂质 | SM-102（或等效MC3） | N/P=6 优化后 |
| 磷脂浓度 | 5.5 mM / 8.8 mM | 两档并行，各有优劣 |
| PEG-脂质 | 1.5 mol% | DSPC为结构磷脂 |
| 积雪草酸 | 38.5 mol%，替代胆固醇 | 文献支持膜融合增强 |
| 目标粒径 | 80~110 nm | 稳定可控 |
| 目标PDI | <0.2（行业标准） | 当前浓缩后可达 |
| 目标包封率 | >80% | 实测 83~93% |

### 1.2 七批实验里程碑

### 1.3 关键数据汇总

**粒径与电位（N/P=6，磷脂8.8mM）：**
- 透析前：~101nm，PDI 0.192，电位+4.6mV
- 浓缩后：~100nm，PDI待测，电位转负

**包封率：**
- 第五批：透析前83.3%，透析后92.6%，浓缩后88.2% ✅
- 第七批：8.8mM组包封损失58.6%（mRNA损失是主要问题）

**转染效率（A549细胞）：**
- AA配方 GFP信号 **明显强于** 传统CHOL配方 ✅
- 这是积雪草酸策略最直接的药效证据

**安全性数据（小鼠）：**
- ≤2.365 mg/kg：✅ 安全
- 3.548 mg/kg：💀 立即死亡
- 安全窗口：LD50 在 2.365~3.548 mg/kg 之间

## 二、ELP多肽修饰质粒体部分（ELP-LNP）

### 2.1 技术原理

ELP = Elastin-Like Polypeptide（弹性蛋白样多肽）

```
LNP核心（SM-102 + DSPC + 积雪草酸 + PEG-lipid）
    ↓
表面偶联 ELP（1%，DMSO预混）
    ↓
在生理环境中形成蛋白冠
    ↓
通过ELP受体（LRP1/Elastin receptor）实现肝外靶向（肺为主）
```

### 2.2 实验批次与数据

**第一批（05.12 新ELP，无mRNA）：**

> ⚠️ 第一批PDI从一开始就高达0.45，根因待确认（ELP批次？缓冲液？有机相配置？）

> ✅ 含mRNA的第四批：浓缩后粒径177nm，PDI 0.138-0.179，包封率88.2%

### 2.3 关键发现

1. **发货批次ELP远优于第一批新ELP** — 缓冲液更换（柠檬酸钠→醋酸钠，05.08）后PDI从0.45降至0.18~0.20，工艺改善明确
2. **浓缩是改善PDI的关键步骤** — 离心浓缩物理打散弱聚集，PDI从0.21~0.25降至0.11~0.14
3. **电位变化符合预期** — 透析前正电位（+10~+19mV）→ 透析后负或接近中性（-1~-2mV），蛋白冠形成机制正常
4. **含mRNA的粒径略大**（177nm vs 120nm）— 正常，mRNA占据内部空间

### 2.4 待解决问题（ELP部分）

| 问题 | 优先级 | 说明 |
|------|--------|------|
| **第一批新ELP的PDI 0.45根因不明** | 🔴 高 | 与发货批次差异巨大，需对照实验确认 |
| **透析后PDI 0.21~0.25 未达标（<0.2）** | 🟡 中 | 依赖浓缩步骤，希望直接达标 |
| **mRNA损失率72.1%** | 🔴 高 | 最大短板，影响成本和实用性 |
| **ELP批次GPC表征未完成** | 🟡 中 | 建议追加，确认ELP分子量分布 |

### 2.5 工艺优化建议（已验证有效措施）

| 措施 | 效果 |
|------|------|
| 醋酸钠缓冲液（50mM，pH=4）| 大幅降低PDI（0.45→0.18）|
| 0.9% NaCl 透析外液 | 渗透压匹配，蛋白冠稳定 |
| 超滤浓缩 | 物理改善PDI（0.24→0.11）|
| DMSO预混ELP后加入 | 避免ELP直接在水相聚集 |

### 2.6 ELP部分结论

> ELP-LNP工艺 **方向正确，但尚有优化空间**。核心成就：建立了积雪草酸+ELP的双机制协同体系（膜融合+靶向），浓缩后PDI可达标。主要剩余问题：①第一批新ELP的PDI根因；②透析后直接达标（不依赖浓缩）；③mRNA回收率提升。
