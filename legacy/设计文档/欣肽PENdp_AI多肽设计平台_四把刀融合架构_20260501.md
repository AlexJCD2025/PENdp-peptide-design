## 一、欣肽PENdp平台定位

| 组件 | 技术 | 阶段 |
|------|------|------|
| **SOTdp** | 仿生胶原透皮 | 研发中 |

**PENdp = 多肽修饰LNP → 肝外靶向（肺）**

核心创新：
- **积雪草酸（AA）** 替代胆固醇 → 增强LNP膜融合，提升细胞摄取
- **DMG-ELP（聚乙二醇化弹性蛋白）** → 蛋白冠机制 → 肺靶向
- **AI驱动序列设计** → 引导扩散模型 → 序列→结构→功能逆向设计

## 三、每把刀在欣肽的具体落地

### 第一关：ML广筛 / 生成式AI

**欣肽的数据库资产：**
- PENdp靶向多肽数据库 v5（21条序列，lung/ngr/irgd等家族）
- 多器官靶向多肽数据库（9大器官 × 50+序列）
- GraphCPP预测分数

**工具链：**
| 工具 | 用途 |
|------|------|
| **ESM-2（8M MPS）** | 多肽序列向量化 |
| **GraphCPP** | CPP（细胞穿透）预测 |
| **ProtGPT2 / RFdiffusion** | 生成全新候选序列（未来扩展）|
| **RDkit** | 理化性质描述符（ClogP、等电点、分子量）|

**过滤条件（欣肽特色）：**

| 过滤项 | 工具/阈值 | 说明 |
|--------|----------|------|
| CPP阳性 | GraphCPP ≥ 0.15 | 必须能穿膜 |
| 分子量 | MW ≤ 5000 Da | 大分子透膜差 |
| 等电点 | 目标范围5-11 | 影响膜穿透 |
| 疏水性 | ClogP 0-3 | 太疏水溶血，太亲水膜穿透差 |
| 溶解性 | pdIPA ≤ 10 | 避免聚集 |

**多目标过滤评分函数（5维度加权）：**
```
Score = 靶向性(35%) + 物化性质(22.5%) + LNP兼容性(18%) + 蛋白冠潜力(18%) + ESM相似度(6.5%)
```

### 第二关：AF3 / 分子对接 精筛

**靶点选择（肺靶向相关）：**

| 靶点 | 机制 | 适合方法 |
|------|------|---------|
| **NRP-1（神经纤毛蛋白）** | iRGD/tLyP-1等靶向 | AF3复合物预测 |
| **CD13（氨肽酶）** | NGR家族靶向 | AF3 + HADDOCK |
| **TGF-β受体** | 胶原合成相关 | 对接 + MM/PBSA |

**积雪草酸的膜融合增强机制：** AA插入膜 → 扰动磷脂双分子层 → 促进LNP内涵体逃逸 → 和对接预测的"膜接近概率"正相关

**输出：** 结合姿态图（机制可视化）+ AlphaFold结构 + 申请专利素材

### 第三关：MD动力学验证

**为什么对欣肽特别重要：**
- 积雪草酸的膜融合是**动态过程**，静态AF3结构只能看稳态
- ELP蛋白冠在血液中会吸附蛋白，复合物构象随时间变化
- 内涵体逃逸效率（k_off）决定了最终转染效果

**建议加入的MD指标：**

| 指标 | 意义 |
|------|------|
| **RMSD轨迹** | 复合物整体稳定性 |
| **RMSF** | 各残基柔性（找不稳定区域）|
| **k_off（解离常数）** |内涵体逃逸效率 |
| **SASA** | 溶剂可及表面积（蛋白冠吸附预测）|
| **氢键寿命** | 多肽-靶点结合持久性 |

### 第四关：QSAR精修

**适用于欣肽的先导肽优化：**

| 先导肽 | 优化方向 |
|--------|---------|
| **iRGD（CRGDKGPDC）** | N端修饰增强NRP-1亲和力 |
| **NGR（CNGRC）** | 环化稳定性 + CD13选择性 |
| **tLyP-1（CGNKRTR）** | C端酰胺化增加蛋白酶耐受性 |

**可用的QSAR工具：**
- **GNN-SAR**（图神经网络）— 不需要手动叠合，适合环肽
- **贝叶斯优化** — 少量实验数据 → 指导下一批合成（Active Learning）
- **Free-Wilson** — 同系列修饰物加和模型

### 第五关：湿实验闭环

**欣肽的湿实验能力：**
- 多肽合成（SPPS）
- LNP制备（微流控）
- 细胞实验（A549转染）
- 动物实验（小鼠毒理）

**数据回流机制：**
```
活性测定结果（IC50/KD）
    ↓
更新训练数据集（标注一致性控制）
    ↓
重新训练ML分类/回归模型
    ↓
新候选生成效率↑
```
