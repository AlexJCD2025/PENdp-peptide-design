## 一、评分体系概述

### 1.1 评分维度

| # | 维度 | 英文 | 权重 |
|---|------|------|------|
| A | 靶向基序 | Targeting Motif | **35%** |
| B | 物化性质 | Physicochemical Properties | **22.5%** |
| C | LNP兼容性 | LNP Compatibility | **18%** |
| D | 蛋白冠形成 | Stealth/Protein Corona | **18%** |
| E | ESM结构相似度 | ESM-2 Structure Similarity | **6.5%** |

**总分 = Σ(A×0.35 + B×0.225 + C×0.18 + D×0.18 + E×0.065)**

每个维度满分100分，最终换算为0-100的综合评分。

## 二、评分细则

### A. 靶向基序（35%）— 最重要

#### A1. 基序数量与特异性（20分）

| 情况 | 得分 |
|------|------|
| 拥有2个及以上明确功能基序（如iRGD的RGD+CendR） | 18-20 |
| 拥有1个明确功能基序（如RGD、NGR、p32结合基序等） | 12-17 |
| 拥有1个弱功能基序或间接靶向序列 | 6-11 |
| 无明确靶向基序 | 0-5 |

#### A2. 基序质量（15分）

| 基序类型 | 得分 | 说明 |
|---------|------|------|
| RGD（αvβ3/β5整合素） | 12 | 经典穿膜靶向基序 |
| NGR（CD13） | 12 | 肿瘤新生血管靶向 |
| CendR（PARPAR/R/K） | 14 | NRP-1介导跨内皮穿透 |
| p32结合基序（AKPC/LinTT1等） | 13 | 肿瘤/M2巨噬靶向 |
| EGFR结合（GE11） | 11 | NSCLC高表达 |
| CD206结合（RP-832c） | 12 | M2型巨噬细胞靶向 |
| 纤维蛋白结合（CREKA） | 10 | 肿瘤归巢 |
| 多靶点组合（iRGD/iNGR双基序） | 额外+3 | 叠加效应 |

**示例计算**：
- iRGD(KD)：RGD(12) + CendR(14) + 多靶点(+3) = 29 → A2得分 = 15（满分）
- AKPC：p32基序(13) = 13 → A2得分 = 13

| 情况 | 得分 |

### D. 蛋白冠形成（18%）

**核心假说**：ELP-LNP表面形成的蛋白冠可被多肽配体协同调控

#### D1. 电荷分布（9分）

| 情况 | 得分 |
|------|------|
| 富含Arg/Lys（正电，促进血清蛋白吸附） | 8-9 |
| 略偏正电（净电荷+1~+3） | 6-7 |
| 中性（净电荷-1~+1） | 4-5 |
| 偏负电（净电荷<-1） | 2-3 |

#### D2. 两亲性特征（9分）

| 情况 | 得分 |
|------|------|
| 明显两亲性（亲水区+疏水区交替） | 8-9 |
| 中等两亲性 | 6-7 |
| 弱两亲性 | 4-5 |
| 无两亲性特征 | 1-3 |

---

### E. ESM结构相似度（6.5%）

**参考序列**：iRGD(KD) — CRGDKGPDC

**计算方法**：
1. 使用ESM-2 8M模型计算每条序列的嵌入向量（1280维）
2. 计算余弦相似度：Sim(iRGD, peptide) = cos(emb_iRGD, emb_peptide)
3. 归一化到0-100分

| 相似度范围 | 得分 |
|-----------|------|
| >0.95 | 6.5 |
| 0.90-0.95 | 5.5 |
| 0.85-0.90 | 4.5 |
| 0.80-0.85 | 3.5 |
| <0.80 | 2.5 |

## 四、评分局限性说明

### 重要提醒

⚠️ **本评分体系为参考框架，非湿实验结论**

1. **靶向基序权重最高（35%）**：依赖文献数据，非直接测量
2. **ESM相似度权重最低（6.5%）**：结构预测有误差
3. **蛋白冠形成机制**：假说阶段，尚需实验验证
4. **GraphCPP CPP概率**：独立维度，衡量"自由穿膜"能力（非PENdp靶向机制的同类指标）

### 建议使用方式

| 目的 | 建议 |
|------|------|
| 初步筛选多肽 | PENdp评分为主，GraphCPP为辅 |
| 设计合成方案 | 参考LNP兼容性维度 |
| 机制研究 | 参考靶向基序详细说明 |
| 向投资人展示 | 用PENdp评分排序，突出Top5 |
