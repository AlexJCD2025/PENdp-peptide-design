## 一、项目概述

### 1.1 PENdp平台定位

| 组件 | 技术 | 阶段 |
|------|------|------|
| **SOTdp** | 仿生胶原透皮 | 研发中 |
| **PENdp** | 核酸递送（积雪草酸+ELP-LNP） | 工艺突破中 |
| **SAPdp** | 自组装药物 | 早期 |

**PENdp = 多肽修饰LNP → 肝外靶向（肺）**

### 1.2 核心创新

- **积雪草酸（AA）** 替代胆固醇 → 增强LNP膜融合，提升细胞摄取
- **DMG-ELP（聚乙二醇化弹性蛋白）** → 蛋白冠机制 → 肺靶向
- **AI驱动序列设计** → 引导扩散模型 → 序列→结构→功能逆向设计

## 四、ESM-2模型配置

### 4.1 模型规格（Mac mini M4 32GB实测）

| 模型 | 参数 | 嵌入维度 | 速度 | 推荐度 |
|------|------|----------|------|--------|
| ESM-2 8M | 8M | 320 | 最快 | ⭐ 快速筛选 |
| ESM-2 35M | 35M | 480 | 快 | ⭐ |
| **ESM-2 150M** | 150M | 640 | **15.9ms/seq** | ⭐⭐⭐ **推荐** |
| **ESM-2 650M** | 650M | 1280 | 38ms/seq | ⭐⭐⭐⭐ **精度最高** |

### 4.2 模型加载代码示例

```python
from transformers import AutoTokenizer, AutoModel
import torch

def load_esm_model(model_size="650M"):
    model_map = {
        "8M": "facebook/esm2_t6_8M_UR50D",
        "35M": "facebook/esm2_t12_35M_UR50D",
        "150M": "facebook/esm2_t30_150M_UR50D",
        "650M": "facebook/esm2_t33_650M_UR50D",
    }
    device = "mps" if torch.backends.mps.is_available() else "cpu"
    model_name = model_map[model_size]
    tokenizer = AutoTokenizer.from_pretrained(model_name)
    model = AutoModel.from_pretrained(model_name).to(device)
    model.eval()
    return model, tokenizer
```

## 五、多肽评分系统

### 5.1 评分函数（5维度加权）

```
Score = 靶向性(35%) + 物化性质(22.5%) + LNP兼容性(18%) + 蛋白冠潜力(18%) + ESM相似度(6.5%)
```

### 5.2 各维度详细说明

| 维度 | 权重 | 评分逻辑 |
|------|------|----------|
| **靶向基序** | 35% | RGD/NGR/CendR等已知基序 |
| **物化性质** | 22.5% | MW ≤ 5000, pI 5-11, GRAVY 0-3 |
| **LNP兼容性** | 18% | 环化、Cys含量、电荷 |
| **蛋白冠形成** | 18% | 两亲性、Arg/Lys富集 |
| **ESM相似度** | 6.5% | 与参考序列的结构相似度 |

# 训练数据：21条多肽（2 CPP+ / 19 Non-CPP）

### 6.2 Top5核心多肽详解

#### iRGD(KD) - 最高优先级

**序列**：H-Cys-Arg-Gly-Asp-Lys-Gly-Pro-Asp-Cys-OH（环肽）  
**分子量**：1093.2 Da  
**靶点**：αvβ3/αvβ5整合素 + NRP-1

**CendR双重靶向机制**：
```
Step 1: RGD → 结合αvβ3/αvβ5（肿瘤/肺血管内皮高表达）
Step 2: K → 被肿瘤微环境中protease（MMP/furin）切割
Step 3: 暴露 CendR motif (R→PARPAR)
Step 4: CendR → 高亲和力结合NRP-1
Step 5: NRP-1 → 跨内皮转运 + 肿瘤深部穿透
Step 6: 进入肿瘤细胞
```

#### iNGR（双靶向新星）

**序列**：H-Cys-Arg-Asn-Gly-Arg-Gly-Pro-Asp-Cys-OH  
**分子量**：1120.2 Da  
**靶点**：CD13（肿瘤血管）+ NRP-1（CendR穿透）= **双靶向**

**优势**：iNGR = NGR靶向CD13 + CendR穿透NRP-1，双机制协同

#### AKPC（p32受体靶向）

**序列**：KPSSPPEE  
**靶点**：CD44/p32受体  
**应用**：LNP修饰专用，ACS Nano 2024验证

## 八、PENdp LNP修饰策略

### 8.1 积雪草酸+ELP-LNP + 多肽靶向 = 完整方案

```
积雪草酸（增强膜融合）
    ↓
ELP（弹性蛋白样多肽，形成蛋白冠）
    ↓
多肽配体（表面修饰）
    ↓
    ├─ iRGD/iNGR → 肿瘤血管+穿透
    ├─ AKPC/LinTT1 → p32+肿瘤/M2巨噬
    ├─ GE11 → NSCLC(EGFR+)
    └─ RP-832c → 肺纤维化(M2)
```

### 8.2 蛋白冠协同假说

```
ELP表面
    ↓ 吸附血清蛋白
蛋白冠形成（富Arg/Lys序列促进）
    ↓
多肽配体 + 蛋白冠协同识别
    ↓
受体介导内吞（CD44/p32/NRP-1/EGFR）
    ↓
积雪草酸增强膜融合 → 核酸胞内释放
```
